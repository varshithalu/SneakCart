## 📌 Overview

SneakCart is a full-stack e-commerce platform designed for online shoe shopping. It enables users to browse products, manage carts and wishlists, authenticate securely, and receive AI-powered shopping assistance through an integrated chatbot.


## 🚀 Features

* User authentication system
* Product browsing interface
* Shopping cart functionality
* Wishlist management
* AI chatbot assistance
* Secure login and registration
* Responsive UI design
* Modern e-commerce experience

---

## 🛠️ Tech Stack

### Frontend
* React.js
* Tailwind CSS
* Redux / Context API

### Backend
* Django

### Database
* MongoDB

### Other Tools
* JWT Authentication
* REST APIs
* GEMINI API

---

## 🧠 Workflow

1. User registers or logs in
2. Products displayed dynamically
3. User adds products to cart or wishlist
4. Cart updates in real time
5. Order processing initiated
6. Chatbot assists users during shopping

## ⚙️ Installation

```bash
git clone https://github.com/your-username/sneakCart.git
cd sneakCart
```

## Frontend
```bash
cd frontend
npm install
npm run dev
```

## Backend
```bash
cd Backend
python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
